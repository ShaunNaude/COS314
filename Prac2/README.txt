1)How to test Program: 
	1.1) ->open a terminal in a GNU/linux system in the project dir.

	1.2) ->execute "$make run"
	      -> you will be prompted to enter the textfile containing the GA parameters (GA.txt).
	      -> you will be prompted to enter the textfile containing the puzzle.

	1.4) NOTE* when testing the difficult puzzles they may take ~4min to complete.

	1.5) The way the provided makefile works is it will place the executable in the bin folder once compiled.

////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

2)Format of GA parameters textfile:
	2.1) -> The GA parameters textfile contains 2 values, the first value is the size of the population
		the second value is the number of generations.

	2.2) -> In the project directory is a file called GA.txt that holds the parameters described above.
